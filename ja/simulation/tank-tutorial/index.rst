
Tankチュートリアル
==================

本チュートリアルでは、Choreonoidのサンプルモデルのひとつである "Tank" モデルをゲームパッド（ジョイスティック）で操作するシミュレーションを実現します。チュートリアルは以下のStep 1からStep 6の6段階で構成されており、これらを順番に実行していくことで、基本的なシミュレーションの実現方法を具体的に学ぶことができます。

   :maxdepth: 3

   step3.rst
   step4.rst
   step5.rst
   step6.rst


.. note:: 本チュートリアルでは4軸以上の操作軸と4つ以上のボタンを備えたゲームパッドが必要となりますが、その実物がない場合でも、Choreonoid本体に含まれる「仮想ジョイスティックビュー」を用いてキーボードで操作することが可能です。

.. note:: 本チュートリアルで作成するプロジェクトやソースコードのファイルは、Choreonoidソースの "sample/tutorial/Tank" 以下に格納されています。
